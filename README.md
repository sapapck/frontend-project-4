# Hexlet Chat (Frontend Project Lvl 4)

Реализация полноценного чата (аналог Slack) с поддержкой каналов, сообщений и авторизации.

### Ссылка на проект:

[Посмотреть вживую на Render](https://frontend-project-4.onrender.com)

---

## Особенности проекта:

- **Стек**: React (Hooks), Redux Toolkit, RTK Query.
- **Интерфейс**: Bootstrap 5 + React-Bootstrap.
- **Реалтайм**: Обновление данных через Socket.io.
- **Валидация**: Formik + Yup.
- **i18n**: Интернационализация на базе i18next (поддержка русского языка).
- **Мониторинг**: Отслеживание ошибок через Rollbar.
- **Стиль**: ESLint (Airbnb) + Prettier.
